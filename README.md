# 小工具箱

一个参考 `Time163 /toolbox` 信息架构重做的静态在线工具网站。

## 站点分析结论

目标站点的核心结构不是单个工具页，而是一个典型的工具门户：

- 左侧固定分类导航
- 顶部搜索、工具数量统计、收藏入口、主题切换
- 首页推荐工具区
- 主体按分类分段展示工具卡片
- 点击卡片后直接进入具体工具
- 底部包含站点介绍与辅助信息

抓取到的 `toolbox` 页包含 14 个一级分类：

1. 推荐工具
2. 生活工具
3. 加密解密
4. 编码转码
5. 格式工具
6. 站长工具
7. 图片颜色
8. 文本工具
9. 数学工具
10. 生成工具
11. PDF 工具
12. 实用工具
13. 健康养生
14. 娱乐工具

## 当前版本实现内容

本版本保留了原站最核心的产品结构，并重新做了视觉与代码组织：

- 响应式左侧分类导航

- 收藏筛选
- 深浅色主题切换

- 分类工具列表
- 统一工具弹层

- 37 个可直接使用的实际工具

## 文件结构

- `index.html`：页面结构
- `styles.css`：视觉样式与响应式布局
- `data.js`：分类、工具与推荐数据
- `app.js`：渲染逻辑、搜索、收藏、主题与工具功能

## 本地预览

直接打开 `index.html` 即可运行。

## 部署建议

这是一个纯静态站点，适合直接部署到 GitHub Pages。
